模块一：PC 电脑的组成

1.1 CPU（中央处理器）

CPU（Central Processing Unit）是计算机的"大脑"，负责执行程序指令、处理数据、控制其他硬件协同工作。其核心工作原理是取指—解码—执行—写回的指令周期（冯·诺依曼架构）。

主要作用：运算逻辑控制、程序指令执行、系统资源管理（内存/IO调度）、多任务并行处理（多核心/超线程）。

1.4 硬盘（存储设备）

硬盘是用于持久化存储数据的设备（断电数据不丢失）。

类型原理优点缺点
HDD 机械硬盘磁头在高速旋转磁盘上读写单位容量便宜，数据可恢复速度慢，怕震动，噪音大
SSD 固态硬盘NAND闪存芯片，无机械部件极快，抗震，静音单位容量贵，写入寿命有限

协议与接口详解

SATA（Serial ATA） — AHCI协议，SATA 3.0主流，理论带宽6Gbps（约550MB/s实际），延迟毫秒级，适用大容量数据仓库盘。

NVMe（Non-Volatile Memory Express） — 专为闪存设计，PCIe通道（M.2物理接口或PCIe插槽），延迟微秒级，适用系统盘/游戏盘/高速缓存。

内存带宽计算公式

带宽 (GB/s) = 频率 (MT/s) × 总线宽度 (位) ÷ 8 ÷ 1000

示例：DDR5-4800（64位总线）：
4800 MT/s × 64 位 ÷ 8 ÷ 1000 = 38.4 GB/s（单通道）
双通道：38.4 × 2 = 76.8 GB/s

电源功率精确计算公式

推荐电源功率 = (CPU TDP × 1.3 + 显卡TDP × 1.2 + 其他硬件功耗) ÷ 0.8

示例：i7-14700K + RTX 4070 SUPER
CPU：125W × 1.3 = 162.5W
显卡：220W × 1.2 = 264W
其他：50W(主板) + 15W(内存) + 10W(SSD) + 15W(风扇) = 90W
小计：162.5 + 264 + 90 = 516.5W
除以 0.8（80%负载率）：516.5 ÷ 0.8 = 645W

→ 推荐：650W 或以上金牌全模组电源
