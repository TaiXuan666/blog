# 模块一：PC 电脑的组成

## 1.1 CPU（中央处理器）

### 作用与工作原理

CPU（Central Processing Unit）是计算机的"大脑"，负责执行程序指令、处理数据、控制其他硬件协同工作。其核心工作原理是**取指—解码—执行—写回**的指令周期（冯·诺依曼架构）。

主要作用：
- 运算逻辑控制（算术/逻辑运算）
- 程序指令执行
- 系统资源管理（内存/IO调度）
- 多任务并行处理（多核心/超线程）

### 核心参数解读

| 参数 | 说明 | 业务选型建议 |
|------|------|-------------|
| 主频（Base/Boost） | CPU每秒时钟周期数，单位GHz，越高单核性能越强 | 游戏/单核应用优先高主频；多核渲染/服务器优先多核心 |
| 核心数/线程数 | 物理核心数/逻辑线程数（超线程技术使1核对应2线程） | 视频剪辑/渲染/虚拟机：多核优先；游戏：6-8核已足够 |
| 缓存（L1/L2/L3） | CPU内置高速存储器，L1最快最小，L3最大但稍慢 | 大缓存对数据库/游戏/科学计算有明显加速作用 |
| TDP（热设计功耗） | CPU在基础频率下的最大发热量，单位W | 选择散热器时需确保散热能力≥TDP，建议留20%余量 |
| 制程工艺 | 晶体管栅极宽度，单位nm，数值越小越先进 | 更先进制程=更高能效比+更低发热，AMD 4nm目前领先 |

## 1.2 显卡（GPU）

### 作用与工作原理

显卡（Graphics Processing Unit，GPU）是专门处理图形和并行计算的处理器。

主要作用：
- **图形渲染**：将3D模型转换为2D图像输出到显示器（游戏/设计）
- **并行计算**：利用数千个小型核心同时处理大量数据（AI训练/科学计算）
- **视频编解码**：硬件加速视频剪辑/转码（Premiere/DaVinci）
- **光线追踪**：实时模拟光线反射/折射（RT Core专用硬件）

## 1.3 内存条（RAM）

### 作用与工作原理

内存（Random Access Memory）是CPU与硬盘之间的中间速度层，用于临时存储正在运行的程序和数据。

主要作用：
- 程序运行载体：所有正在运行的软件都加载在内存中
- 数据缓存：CPU处理的数据先存入内存，再写入硬盘
- 多任务支撑：内存越大，同时运行的程序越多

**重要特性：断电数据丢失（易失性存储）**

---

**关键概念解读：**
- **预取位数（Prefetch）**：每次读取请求可预取的数据量，DDR5的16n使其带宽大幅超越DDR4
- **电压**：每代电压降低，功耗更低，发热更小
- **双通道/四通道**：多个内存条并行工作，带宽成倍增加（需主板支持）

## 1.4 硬盘（存储设备）

### 作用与分类

硬盘是用于持久化存储数据的设备（断电数据不丢失）。

**主要分为两类：**

| 类型 | 全称 | 原理 | 优点 | 缺点 |
|------|------|------|------|------|
| HDD | Hard Disk Drive（机械硬盘） | 磁头在高速旋转的磁盘上读写数据 | 单位容量便宜，数据可恢复 | 速度慢，怕震动，噪音大 |
| SSD | Solid State Drive（固态硬盘） | 闪存芯片存储数据，无机械部件 | 速度快，抗震，静音 | 单位容量较贵，写入寿命有限 |

---

### 协议与接口详解

#### SATA（Serial ATA）

- **协议**：AHCI（Advanced Host Controller Interface）
- **接口**：SATA 3.0（最主流）
- **理论带宽**：6Gbps（约550MB/s实际）
- **延迟**：较高（毫秒级）
- **适用场景**：大容量数据仓库盘、备份盘

#### NVMe（Non-Volatile Memory Express）

- **协议**：NVMe（专为闪存设计，低延迟高并发）
- **接口**：PCIe通道（M.2物理接口或PCIe插槽）
- **理论带宽**：取决于PCIe版本（见下表）
- **延迟**：极低（微秒级）
- **适用场景**：系统盘、游戏盘、高速缓存

### M.2 接口详解

M.2是物理接口规格，支持两种协议：

| 插槽键位 | 支持的协议 | 标识 | 说明 |
|---------|-----------|------|------|
| M-Key（缺口在右） | NVMe（PCIe）×4 | 通常标注"M.2 NVMe" | 高速SSD，目前主流 |
| B-Key（缺口在左） | SATA / PCIe ×2 | 较少见 | 早期M.2 SSD用 |
| B&M-Key（两个缺口） | 兼容SATA和NVMe | 万能兼容 | 早期过渡设计 |

**重要：** 购买M.2 SSD前需确认主板M.2插槽是否支持NVMe协议（部分老主板M.2只支持SATA协议）。

## 1.5 散热系统

### 风冷散热

#### 工作原理
CPU产生热量 → 热量传导至铜基座 → 热管（内部工质气液相变）将热量快速传导至铝鳍片 → 风扇吹过鳍片将热量带走

| 塔式双风扇 | ~160mm | 6-7根 | <250W | 250-450元 | 高端CPU |
| 顶级风冷 | ~165mm | 8根 | <300W | 450-700元 | 高TDP CPU（不超频） |

### 水冷散热

#### 工作原理
CPU产生热量 → 水冷头（吸热）将热量传递给液体 → 液体通过水管流向冷排 → 冷排鳍片将热量散发 → 风扇吹过冷排带走热量 → 冷却后的液体回流至水冷头

## 1.7 电源（PSU）

### 作用与关键参数

电源是PC的"心脏"，将家用交流电（220V）转换为各硬件需要的直流电（+12V/+5V/+3.3V）。

**关键参数解读：**

| 参数 | 说明 | 选型要点 |
|------|------|---------|
| 额定功率 | 电源持续稳定输出的功率，单位W | 必须大于整机峰值功耗，建议留20%余量 |
| 80Plus认证 | 转换效率认证，白牌→铜→银→金→铂金→钛金 | 金牌为性价比甜点，铂金/钛金适合长期开机 |
| 模组化 | 全模组/半模组/非模组 | 全模组线材可自由拆卸，理线更美观 |
| +12V输出 | 现代PC主要功耗在+12V，需关注+12V输出功率 | 高端平台+12V输出应≥整机功耗 |
| ATX 3.0/3.1 | 新版电源规范，支持PCIe 5.0显卡 | RTX 40系列建议选ATX 3.0+电源（原生12VHPWR接口） |
| 单路/双路12V | 单路12V输出更简洁，双路需注意各路功率分配 | 目前主流为单路+12V设计 |

## 2.6 内存条型号及传输速度（详细）

### 内存带宽计算公式

```
带宽(GB/s) = 频率(MT/s) × 总线宽度(位) ÷ 8 ÷ 1000
```

**示例：** DDR5-4800（64位总线）：
- 4800 MT/s × 64位 ÷ 8 ÷ 1000 = **38.4 GB/s（单通道）**
- 双通道：38.4 × 2 = **76.8 GB/s**

### 电源功率计算公式（精确版）

```
推荐电源功率 = (CPU TDP × 1.3 + 显卡TDP × 1.2 + 其他硬件功耗) ÷ 0.8
```

**示例计算（i7-14700K + RTX 4070 SUPER）：**
- CPU：125W × 1.3 = 162.5W
- 显卡：220W × 1.2 = 264W
- 其他：50W（主板）+ 15W（内存）+ 10W（SSD）+ 15W（风扇）= 90W
- 小计：162.5 + 264 + 90 = 516.5W
- 除以0.8（80%负载率）：516.5 ÷ 0.8 = **645W**
- **推荐：650W或以上金牌全模组电源**
